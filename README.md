## About

On my GitHub, I typically release tools that I’ve developed to solve problems I’ve encountered or to practice and experiment with different concepts. Currently, I am a Networking Engineering & Security student, so my main focus is on tool development, and I primarily work with Python and Rust. I try to support both Windows and Linux for my projects.
